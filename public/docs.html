App DESIGN.md

Theme: SPLIT v4, adopted 2026-07-05 from the combined brand board (i2i, sampled hexes). STEMPEL / RASI archived as alternates. Token structure follows the Agentic semantic naming: BACKGROUND / CONTENT / BORDER / INPUT / BUTTON. Every specimen on this page is rendered live from the tokens, not screenshots.

05Atoms

The shipped inventory: same classes, same values as src/App.css. If it renders differently in the app, one of the two is wrong.

Buttons BUTTON / PRIMARY · GHOST · DISABLED
Primary: --color-content-accent fill, --color-content-on-accent label. Ghost: transparent + 1px --color-border-medium. Hover: brightness 1.06.
Input INPUT / TEXT
Focus ring: border-color switches to --color-border-accent, no outline. Click the field to see it.
Pills PILL / STATUS
TESTNET LOW BALANCE
Tint background + matching content color. 11px/700, radius-full.
Code chip CODE / INLINE

13px mono on --color-bg-input, radius-sm, 1px --color-border-medium.
Feedback text TEXT / HINT · ERROR · SUCCESS
